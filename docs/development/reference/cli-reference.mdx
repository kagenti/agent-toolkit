- **Enterprise GitHub**: `https://github.mycompany.com/myorg/myrepo`
- **With a custom Dockerfile location**: `kagenti-adk add --dockerfile /my-agent/path/to/Dockerfile &quot;https://github.com/my-org/my-awesome-agents@main#path=/my-agent&quot;`
