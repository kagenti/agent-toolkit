# Platform Extensions Reference

Use this reference for selecting and implementing Kagenti ADK platform extensions.

## Extension Selection Matrix

| Extension             | Use when the Agent                                                                                       | Documentation                                                                                                                                                                                                                                    |
| --------------------- | -------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **LLM Proxy Service** | Needs platform-provided language model access and credentials                                            | [LLM Proxy Service](https://github.com/kagenti/adk/blob/main/docs/stable/agent-integration/llm-proxy-service.mdx)                                                                                                                                |
| **Forms**             | Requires structured, named parameter inputs (not just free text) as initial input or during conversation | [Collect Input with Forms](https://github.com/kagenti/adk/blob/main/docs/stable/agent-integration/forms.mdx)                                                                                                                                     |
| **Trajectory**        | Yields multi-step reasoning, tool calls, long-running progress, or explicit debugging traces             | [Visualize Agent Trajectories](https://github.com/kagenti/adk/blob/main/docs/stable/agent-integration/trajectory.mdx)                                                                                                                            |
| **Files**             | Needs to read image or document files uploaded by the user                                               | [Working with Files](https://github.com/kagenti/adk/blob/main/docs/stable/agent-integration/files.mdx)                                                                                                                                           |
| **Error**             | Needs to report structured, user-visible failures and stack traces                                       | [Handle Errors](https://github.com/kagenti/adk/blob/main/docs/stable/agent-integration/error.mdx)                                                                                                                                                |
| **Settings**          | Has configurable behavior (for example, "Thinking Mode")                                                 | [Configure Agent Settings](https://github.com/kagenti/adk/blob/main/docs/stable/agent-integration/agent-settings.mdx)                                                                                                                            |
| **OAuth**             | Accesses OAuth-protected third-party APIs (for example, GitHub or Slack)                                 | [OAuth](https://github.com/kagenti/adk/blob/main/docs/stable/agent-integration/oauth.mdx)                                                                                                                                                        |
| **MCP**               | Uses Model Context Protocol tools or servers                                                             | [MCP Integration](https://github.com/kagenti/adk/blob/main/docs/stable/agent-integration/mcp.mdx)                                                                                                                                                |
| **Embedding**         | Performs vector search or uses RAG strategies                                                            | [Build RAG Pipelines](https://github.com/kagenti/adk/blob/main/docs/stable/agent-integration/rag.mdx)                                                                                                                                            |
| **Approval**          | Performs sensitive tool calls requiring user consent                                                     | [Approve Tool Calls](https://github.com/kagenti/adk/blob/main/docs/stable/agent-integration/tool-calls.mdx)                                                                                                                                      |
| **Secrets**           | Needs user-provided API keys or tokens at runtime                                                        | [Manage Runtime Secrets](https://github.com/kagenti/adk/blob/main/docs/stable/agent-integration/secrets.mdx) (Note: Check `secrets.data` and use `request_secrets(params=...)` only through a declared `secrets` extension parameter if missing) |
| **Env Variables**     | Requires custom environment-level deployment configuration variables                                     | [Environment Variables](https://github.com/kagenti/adk/blob/main/docs/stable/agent-integration/env-variables.mdx)                                                                                                                                |
| **Canvas**            | Needs to edit artifacts or code selected by the user                                                     | [Work with Canvas](https://github.com/kagenti/adk/blob/main/docs/stable/agent-integration/canvas.mdx)                                                                                                                                            |
| **Citations**         | References documents or external URLs                                                                    | [Add Citations to Agent Responses](https://github.com/kagenti/adk/blob/main/docs/stable/agent-integration/citations.mdx)                                                                                                                         |

For a complete overview of all available extensions: **[Agent Integration Overview](https://github.com/kagenti/adk/blob/main/docs/stable/agent-integration/overview.mdx)**
